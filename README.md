# HR Analytics Dashboard

## Project Overview

This Tableau dashboard provides an interactive analysis of workforce demographics and employee attrition.

The dashboard helps HR professionals understand workforce composition, identify retention challenges, and make data-driven decisions to improve employee engagement and reduce turnover.

---

## Business Problem

Employee attrition can significantly impact organizational performance and operational costs.

The goal of this dashboard is to answer key HR questions:

- What is the overall attrition rate?
- Which departments experience the highest turnover?
- How does attrition vary across age groups?
- Does attrition differ by gender?
- Which employee groups are at higher risk of leaving?

---

## Dashboard Features

### KPI Overview

The dashboard provides key workforce metrics:

- Total Employees
- Active Employees
- Attrition Count
- Attrition Rate

---

## Visualizations Included

### Employee Distribution by Age Group
Provides workforce demographics across age ranges.

### Attrition by Gender
Compares turnover rates between male and female employees.

### Attrition by Age Group
Identifies age groups with higher attrition rates.

### Department-Wise Attrition
Highlights departments experiencing the greatest employee turnover.

### Education Field Analysis
Examines attrition trends across educational backgrounds.

### Job Satisfaction Analysis
Displays satisfaction ratings across job roles.
